HelloCoakatiel is The activity that has been studied by all 4 parties are Front-end design infra and game.

## Getting Started

Required
- node.js version 10.16.3
- yarn version 1.17.3

### Installing

```
git clone https://github.com/Hello-Cockatiel-2019/HelloCockatiel-PR.git
```
when clone finish

if use yarn
```
yarn 
```

if use npm
```
npm i
```

## Start Project

if use yarn
```
yarn start
```

if use npm
```
npm start
```

## Deployment
```
yarn build
```
